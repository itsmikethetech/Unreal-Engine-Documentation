Caching

Caching

On this page

  • Actions

Actions

Add Asset Reference

Adds a new UObject referencer to a particular asset, returning true if the operation succeeded.

Assets will not be evicted or removed from the cache while the referencer is registered. Note that internally the cache keeps FObjectKey structs constructed from the referencers, instead of direct pointers to them.

Target is USD Asset Cache

Cache Asset

Adds an asset to the cache attached to a particular hash, and optionally registering a referencer

Target is USD Asset Cache

Can Remove Asset

Returns true if the asset with the given hash can be removed from the cache. It will return false in case the asset is still being used, either by another consumer asset or directly by some referencer.

Target is USD Asset Cache

Enable Playback

Enable playback for a specific component using its index in the list of observed component

Target is Chaos Cache Manager

Enable Playback by Cache

Enable playback for a specific component using its cache name

Target is Chaos Cache Manager

Get Cached Asset

Returns an asset associated with a particular Hash. If the asset is persistent, unloaded and the "USD.OnDemandCachedAssetLoading" cvar is true, this may cause the asset to be read from the asset cache's file on disk.

Target is USD Asset Cache

Remove All Asset References

Removes the particular referencer to all assets tracked by the cache, if it was a referencer to any of them. Returns true if at least one asset referencer count was altered by this.

Target is USD Asset Cache

Remove Asset

If an asset is associated with Hash, it will be returned and the asset cache will stop tracking this asset

  1. Returns nullptr otherwise. See CanRemoveAsset.

WARNING: The asset will still be outer'd to the asset cache, however. The caller is in charge of properly placing the asset at a new Outer object.

Target is USD Asset Cache

Remove Asset Reference

Removes an UObject referencer from a particular asset, returning true if the operation succeeded.

Target is USD Asset Cache

Reset All Component Transforms

Resets all components back to the world space transform they had when the cache for them was originally recorded if one is available

Target is Chaos Cache Manager

Reset Single Transform

Resets the component at the specified index in the observed list back to the world space transform it had when the cache for it was originally recorded if one is available

Target is Chaos Cache Manager

Set Cache Collection

change the cache collection for this player if the cache is playing or recording this will have no effect

Target is Chaos Cache Manager

Touch Asset

Marks the provided asset as being used at this point, optionally adding a specific referencer. This is useful because the asset cache will always prioritize retaining the most recently used assets

Target is USD Asset Cache

Trigger All

Triggers the recording or playback of all observed components

Target is Chaos Cache Manager

Trigger Component

Triggers a component to play or record. If the cache manager has an observed component entry for InComponent and it is a triggered entry this will begin the playback or record for that component, otherwise no action is taken.

Target is Chaos Cache Manager

Trigger Component by Cache

Triggers a component to play or record. Searches the observed component list for an entry matching InCacheName and triggers the playback or recording of the linked observed component

Target is Chaos Cache Manager
