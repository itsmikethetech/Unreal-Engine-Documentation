Decode Actions

Call this function when it is time for your agents to take their actions. You most likely want to call this after your policy's EvaluatePolicy function to ensure that you are receiving the latest actions. This will call this agent interactor's GetActions event.

Target is Learning Agents Interactor

Encode Actions

Call this function when it is time to gather all the actions for your agents. This should be called roughly whenever you are calling ULearningAgentsInteractor::EncodeObservations. This will call this controller's SetActions event.

Target is Learning Agents Controller

Encode Observations

Call this function when it is time to gather all the observations for your agents. This can be done each frame or you can consider wiring it up to some kind of meaningful event, e.g. a hypothetical "OnAITurnStarted" if you have a turn-based game. This will call this agent interactor's SetObservations event.

Target is Learning Agents Interactor

End Recording

Ends the recording of the observations and actions of each agent and stores them in the current recording object.

Target is Learning Agents Recorder

End Training

Stops the training process.

Target is Learning Agents Trainer

Evaluate Completions

Call this function when it is time to evaluate the completions for your agents. This should be done at the beginning of each iteration of your training loop after the initial step, i.e. after taking an action, you want to get into the next state before evaluating the completions.

Target is Learning Agents Trainer

Evaluate Critic

Calling this function will run the underlying neural network on the previously buffered observations to populate the output value buffer. This should be called after the corresponding agent interactor's EncodeObservations.

Target is Learning Agents Critic

Evaluate Policy

Calling this function will run the underlying neural network on the previously buffered observations to populate the output action buffer. This should be called after the associated agent interactor's EncodeObservations and before its DecodeActions.

Target is Learning Agents Policy

Evaluate Rewards

Call this function when it is time to evaluate the rewards for your agents. This should be done at the beginning of each iteration of your training loop after the initial step, i.e. after taking an action, you want to get into the next state before evaluating the rewards.

Target is Learning Agents Trainer

Process Experience

Call this function at the end of each step of your training loop. This takes the current observations/actions/ rewards and moves them into the episode experience buffer. All agents with full episode buffers or those which have been signaled complete will be reset. If enough experience is gathered, it will be sent to the training process and an iteration of training will be run and the updated policy will be synced back.
