Unbounded
Boolean

If no Bounding Shape input is provided, the actor bounds are used to limit the sample generation domain. This option allows ignoring the actor bounds and generating over the entire spline. Use with caution as this may generate a lot of points.
Interior Sample Spacing
Float (single-precision)

The space between each sample point
Interior Border Sample Spacing
Float (single-precision)

The space between each sample point on the spline boundary. Used for computation; lower spacing is more expensive but more accurate.
Treat Spline As Polyline
Boolean

Use the spline points to form a polyline, instead of computing many sample points along the spline. This is more accurate if your spline is linear.
Interior Orientation
EPCGSplineSamplingInteriorOrientation Enum

Determines the orientation of interior points
Point Steepness
Float (single-precision)

Each PCG point represents a discretized, volumetric region of world space. The points' Steepness value [0.0 to 1.0] establishes how "hard" or "soft" that volume will be represented. From 0, it will ramp up linearly increasing its influence over the density from the point's center to up to two times the bounds. At 1, it will represent a binary box function with the size of the point's bounds.
