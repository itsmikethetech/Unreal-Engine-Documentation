Inputs

 Render Data Identifier
Movie Graph Render Data Identifier Structure

This is used to fill out tokens related to the render data that the file represents. Used to fill out the {camera_name}, {render_layer}, {renderer_name}, {renderer_sub_name} tokens.
Root Frame Number
Integer

Frame Number for the Root (matching what you see in the Sequencer timeline. ie: If the Sequence PlaybackRange starts on 50, this value would be 50 on the first frame. Used to fill out the {frame_number} token.
Shot Frame Number
Integer

Frame Number for the Shot (matching what you would see in Sequencer at the sub-sequence level. Used to fill out the {frame_number_shot} token.
Root Frame Number Rel
Integer

Frame Number for the Root (relative to 0, not what you would see in the Sequencer timeline. ie: If sequence PlaybackRange starts on 50, this value would be 0 on the first frame.) Used to fill out the {frame_number_rel} token.
Shot Frame Number Rel
Integer

Frame Number for the Shot (relative to 0, not what you would see in the Sequencer timeline.) Used to fill out the {frame_number_shot_rel} token.
File Metadata
Map of Strings to Strings

A key/value pair that maps metadata names to their values. Output is only supported in exr formats at the moment.
Version
Integer

Used to resolve {version} format arguments.
Force Relative Frame Numbers
Boolean

If true, force format strings (like {frame_number}) to resolve using the relative version. Used when slow-mo is detected as frame numbers would overlap.
Ensure Absolute Path
Boolean

If true, If the format string is a relative path, then the resulting resolved path will be converted to an absolute path. Assumes that the relative path should be resolved relative to the engine/editor executable.
Initialization Time
Date Time Structure

The initialization time for this job. Used to resolve time-based format arguments.
Zero Pad Frame Number Count
Integer

When converitng frame numbers to strings, how many digits should we pad them up to? ie: 5 => 0005 with a count of 4.
File Name Format Overrides
Map of Strings to Strings

A map between "{format}" tokens and their values. These are applied after the auto-generated ones from the system, which allows the caller to override things like {.ext}
Job
Movie Pipeline Executor Job Object Reference

  1. If specified, used to resolve some tokens related to the job.

Shot
Movie Pipeline Executor Shot Object Reference

  1. If specified, used to resolve the shot name. No settings from this shot are ever used (they must come form the EvaluatedConfig instead).

Evaluated Config
Movie Graph Evaluated Config Object Reference

  1. If specified, used to resolve some tokens related to settings in the shot (such as {ts_count}).

Frame Number Offset
Integer

Added onto the various FrameNumber settings automatically. Can be used to offset values (such as starting your Level Sequence on frame 0, but outputting it to disk at 1001).
Default Frame Rate
Frame Rate Structure

What is the default Frame Rate? This will be overwritten by the Output Setting from the evaluated config (if specified).
