UTransformGizmo

[UTransformGizmo](API\Editor\EditorInteractiveToolsFramework\EditorGizmos\UTransformGizmo) provides standard Transformation Gizmo interactions, applied to a [UTransformProxy](API\Runtime\InteractiveToolsFramework\BaseGizmos\UTransformProxy) target object.

Header

/Engine/Source/Editor/Experimental/EditorInteractiveToolsFramework/Public/EditorGizmos/TransformGizmo.h

Include

#include "EditorGizmos/TransformGizmo.h"

Syntax

class UTransformGizmo :
    public UInteractiveGizmo,
    public IHoverBehaviorTarget,
    public IClickDragBehaviorTarget

Remarks

UTransformGizmo provides standard Transformation Gizmo interactions, applied to a UTransformProxy target object. By default the Gizmo will be a standard XYZ translate/rotate Gizmo (axis and plane translation).
