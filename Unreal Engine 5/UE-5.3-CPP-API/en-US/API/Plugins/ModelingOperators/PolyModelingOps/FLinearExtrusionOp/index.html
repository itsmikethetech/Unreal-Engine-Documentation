FLinearExtrusionOp

LinearExtrusionOp extrudes a set of triangles of a mesh in a linear direction, with various options for how to process that extrusion.

class FLinearExtrusionOp : public UE::Geometry::FDynamicMeshOperator

Remarks

LinearExtrusionOp extrudes a set of triangles of a mesh in a linear direction, with various options for how to process that extrusion.
