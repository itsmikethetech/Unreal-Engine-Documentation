UScriptableInteractiveTool

[UScriptableInteractiveTool](API\Plugins\ScriptableToolsFramework\UScriptableInteractiveTool) is an extension of [UInteractiveTool](API\Runtime\InteractiveToolsFramework\UInteractiveTool) that allows the Tool functionality to be defined via Blueprints.
