UPoseSearchLibrary::MotionMatch

Implementation of the core motion matching algorithm

PoseSearch

Include

Source

/Engine/Plugins/Experimental/Animation/PoseSearch/Source/Runtime/Private/PoseSearchLibrary.cpp

Syntax

UFUNCTION(BlueprintPure, Category="Animation|Pose Search",
          Meta=(BlueprintThreadSafe, Keywords="PoseMatch"))
static void MotionMatch
(
    UAnimInstance * AnimInstance,
    const UPoseSearchDatabase * Database,
    const FPoseSearchQueryTrajectory Trajectory,
    float TrajectorySpeedMultiplier,
    const FName PoseHistoryName,
    UAnimationAsset *& SelectedAnimation,
    float & SelectedTime,
    bool & bLoop,
    bool & bIsMirrored,
    FVector & BlendParameters,
    float & SearchCost,
    const UAnimationAsset * FutureAnimation,
    float FutureAnimationStartTime,
    float TimeToFutureAnimationStart,
    const int DebugSessionUniqueIdentifier
)

Remarks

Implementation of the core motion matching algorithm

Parameters

Parameter

Description

AnimInstance

Input animation instance

Database

Input database to search

Trajectory

Input motion trajectory samples for pose search queries. Expected to be in the space of the SkeletalMeshComponent. This is provided with the CharacterMovementTrajectory Component output.

TrajectorySpeedMultiplier

Input Trajectory velocity will be multiplied by TrajectorySpeedMultiplier: values below 1 will result in selecting animation slower than requested from the original Trajectory

PoseHistoryName

Input tag of the associated PoseSearchHistoryCollector node in the anim graph

SelectedAnimation

Output selected animation from the Database asset

SelectedTime

Output selected animation time

bLoop

Output selected animation looping state

bIsMirrored

Output selected animation mirror state

BlendParameters

Output selected animation blend space parameters (if SelectedAnimation is a blend space)

SearchCost

Output search associated cost

FutureAnimation

Input animation we want to match after TimeToFutureAnimationStart seconds

FutureAnimationStartTime

Input start time for the first pose of FutureAnimation

TimeToFutureAnimationStart

Input time in seconds before start playing FutureAnimation (from FutureAnimationStartTime seconds)

DebugSessionUniqueIdentifier

Input unique identifier used to identify TraceMotionMatchingState (rewind debugger / pose search debugger) session. Similarly the MM node uses Context.GetCurrentNodeId()
