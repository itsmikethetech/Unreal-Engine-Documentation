Model

A [Model](API\Plugins\MutableRuntime\MuR\Model) represents a customisable object with any number of parameters.

References

Module

MutableRuntime

Header

/Engine/Plugins/Experimental/Mutable/Source/MutableRuntime/Public/MuR/Model.h

Include

#include "MuR/Model.h"

Syntax

class Model

Remarks

When values are given to the parameters, specific Instances can be built, which hold the built application-usable data.
