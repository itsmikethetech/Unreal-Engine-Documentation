Syntax

class RefCounted : public mu::Base

Remarks

Base class for all reference counted objects. Any subclass of this class can be managed using smart pointers through the Ptr<T> template.

This base allow multi-threaded manipulation of smart pointers, since the count increments and decrements are atomic.
