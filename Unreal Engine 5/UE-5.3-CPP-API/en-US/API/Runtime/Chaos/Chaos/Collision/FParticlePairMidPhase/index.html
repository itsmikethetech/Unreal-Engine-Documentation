FParticlePairMidPhase

Produce collisions for a particle pair A [FParticlePairMidPhase](API\Runtime\Chaos\Chaos\Collision\FParticlePairMidPhase) object is created for every particle pair whose bounds overlap. It is responsible for building a set of potentially colliding shape pairs and running collision detection on those pairs each tick.

Remarks

The lifetime of midphase objects is handled entirely by the CollisionConstraintAllocator. Nothing outside of the CollisionConstraintAllocator should hold a pointer to the detector or any constraints it creates for more than the duration of the tick.
