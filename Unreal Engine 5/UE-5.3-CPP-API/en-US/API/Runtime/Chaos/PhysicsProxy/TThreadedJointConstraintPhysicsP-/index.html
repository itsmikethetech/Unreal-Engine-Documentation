TThreadedJointConstraintPhysicsProxyBase

Wrapper class that routes all reads and writes to the appropriate joint data.

Inheritance Hierarchy

IPhysicsProxyBase

FJointConstraintPhysicsProxy

TThreadedJointConstraintPhysicsProxyBase

References

Module

Chaos

Header

/Engine/Source/Runtime/Experimental/Chaos/Public/PhysicsProxy/JointConstraintProxy.h

Include

#include "PhysicsProxy/JointConstraintProxy.h"

Syntax

template<bool bExternal>
class TThreadedJointConstraintPhysicsProxyBase : protected Chaos::FJointConstraintPhysicsProxy

Remarks

Wrapper class that routes all reads and writes to the appropriate joint data. This is helpful for cases where we want to both write to a joint and a network buffer for example
