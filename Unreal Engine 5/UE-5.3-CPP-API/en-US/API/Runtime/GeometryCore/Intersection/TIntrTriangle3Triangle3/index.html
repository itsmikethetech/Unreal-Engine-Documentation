TIntrTriangle3Triangle3

Compute intersection between 3D triangles use [Test()](API\Runtime\GeometryCore\Intersection\TIntrTriangle3Triangle3\Test) for fast boolean query, does not compute intersection info use [Find()](API\Runtime\GeometryCore\Intersection\TIntrTriangle3Triangle3\Find) to compute full information [By](API\Developer\AutomationDriver\By) default fully-contained co-planar triangles are not reported as intersecting.

Remarks

Compute intersection between 3D triangles use Test() for fast boolean query, does not compute intersection info use Find() to compute full information By default fully-contained co-planar triangles are not reported as intersecting. Call SetReportCoplanarIntersection(true) to handle this case (more expensive)
