UStreamableSparseVolumeTexture

Represents a streamable SparseVolumeTexture asset and serves as base class for [UStaticSparseVolumeTexture](API\Runtime\Engine\SparseVolumeTexture\UStaticSparseVolumeTexture) and [UAnimatedSparseVolumeTexture](API\Runtime\Engine\SparseVolumeTexture\UAnimatedSparseVolumeTexture). It has an array of [USparseVolumeTextureFrame](API\Runtime\Engine\SparseVolumeTexture\USparseVolumeTextureFrame).
