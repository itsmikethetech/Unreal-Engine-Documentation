FPreparedType

Represents the prepared type of an [FExpression](API\Runtime\Engine\HLSLTree\FExpression).

Syntax

class FPreparedType

Remarks

Represents the prepared type of an FExpression. Each FExpression will prepare a type during FExpression::PrepareValue(). The prepared type is represented by a concrete Shader::FType, along with a list of FPreparedComponents. This structure matches FRequestedType, and in fact FRequestedType is often used as a 'mask' to only consider certain prepared components
