FNetBitArray

Simple bit array with internal storage.

Inheritance Hierarchy

FNetBitArrayBase

FNetBitArray

References

Module

NetCore

Header

/Engine/Source/Runtime/Net/Core/Public/Net/Core/NetBitArray.h

Include

#include "Net/Core/NetBitArray.h"

Syntax

class FNetBitArray : public UE::Net::FNetBitArrayBase

Remarks

Simple bit array with internal storage. Has very little error checking as it is used in performance critical code.
